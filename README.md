# Creative Portfolio (Starter)

A responsive static portfolio template inspired by a purple "creative portfolio" design.

## Files
- `index.html` — main markup with placeholders
- `styles.css` — styles and responsive layout
- `script.js` — small UI interactions
- `README.md` — this file

## How to run locally (PowerShell)
1. Open PowerShell in the project folder (e.g., `c:\Users\Ron Justine\Desktop\Portfolio`).
2. Run a simple HTTP server:

Option A — Python (if installed):
```powershell
python -m http.server 8000
Start-Process "http://localhost:8000"
